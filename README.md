Configuring git
---------------

  1. Login with `gh`:

  ```
  $ gh auth login
  ```

  2. Create personal access token for machine by going to `https://github.com/settings/tokens` and paste it in `~/.config/hub`

  ```
  github.com:
   - user: cgeorgii
     oauth_token: PASTE_TOKEN_HERE
     protocol: https
  ```
